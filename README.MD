# My AdminDashboard

## Available Scripts

### Step 1: Install Node.js 20.5.0 using Node Version Manager (nvm)

```bash
    nvm install 20.5.0
```

### Step 2: Switch to Node.js 20.5.0
```bash
    nvm use 20.5.0
```

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### Running the production server.

```bash
    npm run start
```
